# WireGuard VPN on AWS EC2

A beginner-friendly project that demonstrates how to build a personal VPN using WireGuard on an AWS EC2 Ubuntu instance. This project focuses on understanding VPN architecture, Linux networking, and secure tunneling.

## Tech Stack

- AWS EC2
- Ubuntu 24.04 LTS
- WireGuard
- Bash
- Linux Networking

## Project Roadmap

- [x] Launch EC2 Instance
- [x] Connect via SSH
- [x] Install WireGuard
- [x] Generate Server Keys
- [ ] Configure WireGuard Server
- [ ] Enable IP Forwarding
- [ ] Configure Firewall (NAT)
- [ ] Configure Android Client
- [ ] Test VPN Connection
- [ ] Documentation & Cleanup

## Learning Goals

- Understand how VPNs work
- Learn WireGuard fundamentals
- Configure secure tunnels
- Explore Linux networking
- Deploy infrastructure on AWS
