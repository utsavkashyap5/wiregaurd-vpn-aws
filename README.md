# WireGuard VPN on AWS EC2

A beginner-friendly project that sets up a personal VPN using WireGuard on an AWS EC2 Ubuntu instance.

## Tech Stack

- AWS EC2
- Ubuntu 24.04
- WireGuard
- Bash
- Linux Networking
